# Prerequisites: Rust and Node.js Installation

Before installing the Cedra CLI and developing on the Cedra network, you'll need to set up two essential development tools: **Rust** and **Node.js**.

- **Rust**: Required for building Cedra from source, developing Move modules, and working with the Cedra blockchain core infrastructure
- **Node.js**: Essential for DApp development, running JavaScript/TypeScript tools, and interacting with Cedra's frontend ecosystem

### Version Compatibility

For optimal compatibility with Cedra we recommend to use:
- **Rust**: 1.70.0 or higher
- **Node.js**: 18.x LTS or 20.x LTS
- **npm**: 8.0.0 or higher

## Rust Installation

:::note
- **Supported Platforms**: Windows 10/11 (64-bit), macOS 10.12+, Linux (kernel 3.2+, glibc 2.17+)
:::

## Node.js Installation

### Recommended Version

Node.js 18.x LTS or 20.x LTS

### Installation

<Tabs groupId="operating-systems">
<TabItem value="macos" label="macOS">

#### Using Homebrew (Recommended)

```bash
brew install node
```

#### Using Node Version Manager (nvm)

1. Install nvm:
   ```bash
   curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
   ```

2. Reload shell configuration:
   ```bash
   source ~/.bashrc
   ```

3. Install Node.js:
   ```bash
   nvm install --lts
   nvm use --lts
   ```

#### Using Official Installer

Download the LTS version from [nodejs.org](https://nodejs.org/) and run the installer.

</TabItem>
<TabItem value="windows" label="Windows">

#### Using Official Installer (Recommended)

1. Download the LTS version from [nodejs.org](https://nodejs.org/)
2. Run the installer
3. Ensure "Add to PATH" is checked during installation

#### Using Chocolatey

```powershell
choco install nodejs
```

#### Using Node Version Manager (nvm-windows)

1. Download and install [nvm-windows](https://github.com/coreybutler/nvm-windows/releases)
2. Open terminal as Administrator:
   ```powershell
   nvm install lts
   nvm use lts
   ```

</TabItem>

### Post-Installation Setup

1. **Update npm to latest version:**
   ```bash
   npm install -g npm@latest
   ```

2. **Configure npm permissions (Linux/macOS only):**
   ```bash
   mkdir ~/.npm-global
   npm config set prefix '~/.npm-global'
   echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
   source ~/.bashrc
   ```

3. **Install common global packages:**
   ```bash
   npm install -g yarn pnpm typescript
   ```

### Verification
