Download and run the official installer



## üì¶ Installing Node.js

Time to set up your JavaScript runtime! We recommend Node.js 18.x LTS or 20.x LTS for maximum stability.

<Tabs groupId="operating-systems">
<TabItem value="macos" label="üçé macOS">

#### Homebrew (Lightning Fast)

```bash
brew install node
```

#### Node Version Manager (For the Power Users)

1. Get nvm up and running:
   ```bash
   curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
   ```

2. Refresh your shell:
   ```bash
   source ~/.bashrc
   ```

3. Install the latest LTS:
   ```bash
   nvm install --lts
   nvm use --lts
   ```

#### Traditional Installer

Prefer clicking? Download from [nodejs.org](https://nodejs.org/) and follow the wizard!

</TabItem>
<TabItem value="windows" label="ü™ü Windows">

Official Installer

1. Head to [nodejs.org](https://nodejs.org/)
2. Download the LTS version
3. Run the installer (keep "Add to PATH" checked!)

Chocolatey Magic

```powershell
choco install nodejs
```

Version Manager (nvm-windows)

1. Download [nvm-windows](https://github.com/coreybutler/nvm-windows/releases)
2. Install it, then run as Administrator:
   ```powershell
   nvm install lts
   nvm use lts
   ```

</TabItem>
<TabItem value="linux" label="üêß Linux">

#### NodeSource Repository (Recommended)

```bash
# Add NodeSource repo and install
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
```

#### Node Version Manager (Flexible Option)

1. Install nvm:
   ```bash
   curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
   ```

2. Reload your shell config:
   ```bash
   source ~/.bashrc
   ```

3. Install Node.js:
   ```bash
   nvm install --lts
   nvm use --lts
   ```

### üîß Final Touches

1. **Supercharge npm:**
   ```bash
   # Always use the latest npm
   npm install -g npm@latest
   ```

2. **Fix npm permissions (Linux/macOS):**
   ```bash
   # Create a safe space for global packages
   mkdir ~/.npm-global
   npm config set prefix '~/.npm-global'
   echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
   source ~/.bashrc
   ```

3. **Install developer essentials:**
   ```bash
   # The holy trinity of JavaScript tools
   npm install -g yarn pnpm typescript
   ```
